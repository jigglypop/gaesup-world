# 4.5 트래커 포텐셜 수치 프로토타입(ε 유도 가설)

## 1. 목적

- Quintessence형 트래커 $V(\Phi)=\Lambda^{4+n}/\Phi^n$ (혹은 $V=V_0 e^{-\lambda\Phi/M_P}$)에서 오늘 $\Omega_\Phi(a=1)$가 보편적으로 $\approx0.685$에 수렴하는지 검증해 $\epsilon_{\text{mass}}=2\Omega_\Lambda-1\approx0.37$를 **무튜닝 수렴값**으로 얻을 수 있는지 수치 점검.

## 2. 동역학 방정식(표준)

$$
\ddot\Phi+3H\dot\Phi+V'(\Phi)=0,\qquad H^2=\frac{8\pi G}{3}(\rho_m+\rho_r+\rho_\Phi),\quad \rho_\Phi=\tfrac12\dot\Phi^2+V(\Phi).
$$

상태변수:
$$
x\equiv \frac{\dot\Phi}{\sqrt{6}H M_P},\quad y\equiv \frac{\sqrt{V}}{\sqrt{3}H M_P},\quad \lambda(\Phi)\equiv -M_P\frac{V'}{V}.
$$

자율계(점근 표준형):
$$
\frac{dx}{dN}=-3x+\sqrt{\tfrac32}\,\lambda y^2+\tfrac32 x\,[1+w_b(1-x^2-y^2)],\quad \frac{dy}{dN}=-\sqrt{\tfrac32}\,\lambda xy+\tfrac32 y\,[1+w_b(1-x^2-y^2)],
$$
$$
\Omega_\Phi=x^2+y^2,\quad w_\Phi=\frac{x^2-y^2}{x^2+y^2},\quad w_b=0\,(\text{물질시대}).
$$

트래커 후보:
- 역멱형: $\lambda(\Phi)=n M_P/\Phi$ (진화형)
- 지수형: $\lambda=\text{const}$ (상수)

## 3. 검증 절차(무튜닝 규칙)

1) 포텐셜 형태를 **사전 고정**(예: $n\in\{1,2,3\}$ 또는 $\lambda\in\{0.1,0.5,1.0\}$).
2) 넓은 초기값(\,${x_\text{ini}},y_\text{ini}}$\,) 샘플에서 $N\equiv\ln a$를 $-7\to0$까지 적분.
3) 오늘 $\Omega_\Phi(0)$가 초기조건과 무관하게 한 값으로 수렴하는지 확인.
4) 수렴값이 $\Omega_\Phi(0)\approx0.685$이면 $\epsilon_{\text{mass}}\equiv 2\Omega_\Lambda-1\approx0.37$ **자연 유도**.

## 4. 파이썬 수치 셀(프로토타입)

```python
import numpy as np

def step_tracker(x, y, lam, w_b=0.0):
    # dx/dN, dy/dN for matter era
    dx = -3*x + np.sqrt(1.5)*lam*y*y + 1.5*x*(1 + w_b*(1 - x*x - y*y))
    dy = -np.sqrt(1.5)*lam*x*y + 1.5*y*(1 + w_b*(1 - x*x - y*y))
    return dx, dy

def integrate_tracker(x0, y0, lam_or_n, Nmin=-7.0, Nmax=0.0, nsteps=5000, model='exp'):
    N = np.linspace(Nmin, Nmax, nsteps)
    x = np.zeros_like(N); y = np.zeros_like(N)
    x[0]=x0; y[0]=y0
    for i in range(nsteps-1):
        lam = lam_or_n if model=='exp' else (lam_or_n) # placeholder for n-dependent λ(Φ)
        dx, dy = step_tracker(x[i], y[i], lam)
        h = N[i+1]-N[i]
        x[i+1] = x[i] + h*dx
        y[i+1] = y[i] + h*dy
        # enforce 0≤Ω≤1
        om = x[i+1]**2 + y[i+1]**2
        if om>1: 
            x[i+1] /= np.sqrt(om); y[i+1] /= np.sqrt(om)
    return N, x, y

for lam in [0.1, 0.5, 1.0]:
    N,x,y = integrate_tracker(1e-4, 1e-6, lam, model='exp')
    Om = x[-1]**2 + y[-1]**2
    print(lam, Om, 2*Om - 1)  # -> target Ω≈0.685, ε≈0.37
```

주: 역멱형의 경우 $\lambda(\Phi)$ 갱신식이 필요(추가 구현 항목).

## 5. 판정 기준

- 초기조건 다양성에도 $\Omega_\Phi(0)$이 안정 수렴: PASS(무튜닝), 비수렴: FAIL.
- 수렴값 $\approx0.685$: ε 자연 유도 성공 시그널.


## 7. 분석 보강: 역멱형의 $w_\Phi$와 관측과의 정합

물질시대 트래킹 해의 근사식(표준 결과):
$$
w_\Phi \simeq -\frac{2}{n+2},\qquad (V\propto \Phi^{-n},\; w_b=0).
$$
관측 $w\approx -1.03\pm0.03$를 만족하려면 $n\ll1$이 요구되어 사실상 $\Lambda$CDM에 근접. $n=1\Rightarrow w_\Phi\approx -2/3$ (관측과 불일치), $n=2\Rightarrow -0.5$, $n=3\Rightarrow -0.4$.

결론: 역멱형은 $\Omega_\Phi\to0.685$ 수렴 가능성은 있으나 $w\approx -1$을 동시 만족하기 어렵다. 가속·정밀 $w$를 충족하려면 **평탄한 포텐셜(PNGB/plateau)** 류가 유리하며, 그 경우에도 포텐셜 형태는 **사전 고정**되어야 한다(무튜닝 원칙).
