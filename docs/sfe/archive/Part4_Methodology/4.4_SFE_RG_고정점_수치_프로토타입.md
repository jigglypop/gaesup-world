# 4.4 RG 고정점 수치 프로토타입(가설 검증용)

## 1. 목적과 가정

- 목적: $\epsilon = g_B\mu/\sqrt{\lambda}$가 적외선(IR) 흐름에서 **고정점** $c_*$로 수렴한다는 가설의 수치적 실험 구조 제시.
- 가정: (i) 무튜닝 — 초기조건 의존성이 사라지는 안정 고정점, (ii) 섹터 분리 유지, (iii) 파라미터 선택은 사전 고정(데이터-피팅 금지).

## 2. 토이 베타함수(사전 고정)

$$
\beta_\epsilon \equiv \frac{d\epsilon}{d\ln\mu_{\rm RG}} = a\,\epsilon\Big(1-\frac{\epsilon}{c_*}\Big),\qquad a>0,\; c_*\in(0,1).
$$

- 해: $\epsilon(t)=\dfrac{c_*}{1+K e^{-a t}}$ ($t\equiv\ln\mu_{\rm RG}$), 모든 초기조건에 대해 $t\to-\infty$에서 $\epsilon\to c_*$.
- 사전 고정: $a=1$ (단위 선택), $c_*=0.37$ (목표 고정점 가설) — 데이터로부터가 아니라 “검증 대상 값”으로 고정.

주: 본 장은 **고정점 존재 시** 무튜닝 수렴이 가능한지를 보이는 구조 제시이며, $c_*$ 유도는 15장에서 논의.

## 3. 수치 셀(검증 하네스)

```python
import numpy as np

def flow_eps(eps0, a=1.0, cstar=0.37, tmin=-10, tmax=0, n=1000):
    t=np.linspace(tmin,tmax,n)
    # analytic solution: eps(t)= c*/(1+K e^{-a t}), K=c*/eps0 - 1
    K = cstar/eps0 - 1.0
    eps = cstar/(1.0 + K*np.exp(-a*t))
    return t, eps

for e0 in [0.05, 0.2, 0.8]:
    t, eps = flow_eps(e0)
    print(e0, float(eps[0]), float(eps[-1]))  # -> (~c*, c*)
```

예상: 초기조건과 무관하게 $\epsilon\to c_*$ 수렴.

## 4. 적용 메모

- 미시 배율과의 정합: $\epsilon\to c_*$이면 결맞음·LIGO·뮤온 배율이 자동 고정.
- 거시와의 연결: $\epsilon_{\text{grav}}$는 별도의 함수형 $\mu(a)$로 다룸(섹터 분리 유지).
