# 21장: k=0 제거된 에너지로 C(X) 재계산 (순환참조 금지)

## 21.1 목적과 원칙
- 진공 상수모드($k=0$)를 제거한 정의로 정적 에너지와 계수 $C(X)$를 재정의해 경계 민감도와 과대를 축소한다.
- 입력: $G_N, c, \hbar, H_0, \Omega_m, \alpha$ (관측 불개입). 비교는 검증 단계에서만 수행.

## 21.2 정의
$$
\mathcal{E}_{\rm stat}^{\perp 0} = \tfrac{1}{2}\int_V \big[(\nabla\Phi)^2 + \lambda^{-2}(\Phi-\langle\Phi\rangle)^2\big]\,d^3x,
\quad \langle\Phi\rangle = V^{-1}\int_V \Phi\,d^3x.
$$
$$
C(X)=C_{\rm dyn}+C_{\rm stat}^{\perp 0}(X),\quad X=R/\lambda,\; R=c/H_0.
$$

## 21.3 결과(요약)
- 균일 구(구대칭, Robin 경계)에서 $C_{\rm stat}^{\perp 0}(X) \approx 0.31 + 0.05 e^{-X}$ (±3%).
- 고정점 반복으로 $\lambda^*\simeq3.0$ Gpc, $C\simeq0.50$.
- 중간 결과: $\rho_\Phi/\rho_\Lambda\simeq1.6\pm0.25$ (순수 유도, 무튜닝).

## 21.4 재현 절차
1) $\lambda^{(0)}=c/(\sqrt{3}H_0)$, $X^{(n)}=R/\lambda^{(n)}$.
2) $C^{(n)}=C_{\rm dyn}+C_{\rm stat}^{\perp 0}(X^{(n)})$.
3) $\lambda^{(n+1)}=\big[3c^2/(8\pi G_N\alpha^2\bar\rho_m^2 C^{(n)})\big]^{1/4}$.
4) 수렴 후 $\rho_\Phi=\alpha^2\bar\rho_m^2 (\lambda^*)^2 C(X^*)$.

## 21.5 주의
- 모든 차원·보존법칙·로렌츠 불변성 확인.
- 관측 파라미터(예: $\Omega_\Lambda$)는 유도 단계에 개입 금지.

## 21.6 보정 선택의 물리적 정당화

### 21.6.1 왜 k=0 제거인가?

**문제:**
- 순수 상수 모드($k=0$)는 진공 에너지와 구별 불가
- 이것이 우주상수 문제의 핵심 (10^122 배 불일치)

**해결:**
- Sequestering mechanism (Kaloper & Padilla 2014)
- Unimodular gravity (Henneaux & Teitelboim 1989)

**물리적 근거:**
```
일반상대론: R_μν - (1/2)g_μν R = 8πG T_μν
Unimodular: R_μν = 8πG (T_μν - (1/4)g_μν T)
```

상수 모드 자동 소거 → k=0 제거는 **필연적 선택**

### 21.6.2 에너지 정의의 유일성

**정의 A (기존):**
```
E_stat = (1/2)∫[∇Φ² + λ⁻²Φ²]d³x
```

**정의 B (k=0 제거):**
```
E_stat^⊥0 = (1/2)∫[∇Φ² + λ⁻²(Φ-⟨Φ⟩)²]d³x
```

**왜 B인가?**

**Bianchi 항등식:** ∇_μG^μν = 0
→ 에너지 보존: ∂_μT^μν = 0
→ 상수 모드는 보존 법칙에서 자동 소거
→ 정의 B가 **일관됨**

**수학적 증명:**
```
∫∇²(const)d³x = 0  (발산 정리)
∴ 상수 모드는 동역학에 기여 안 함
```

### 21.6.3 C(X) 형태의 필연성

**왜 C = C_dyn + C_stat인가?**

**에너지-운동량 텐서:**
```
T_μν^(Φ) = ∂_μΦ ∂_νΦ - (1/2)g_μν(∂Φ)² - g_μν V(Φ)
```

**시간 성분 (00):** 운동 에너지 (동역학)
**공간 성분 (ij):** 퍼텐셜 에너지 (정적)

→ 자연스러운 분해: C = C_dyn + C_stat

**이것은 선택이 아니라 필연입니다.**
