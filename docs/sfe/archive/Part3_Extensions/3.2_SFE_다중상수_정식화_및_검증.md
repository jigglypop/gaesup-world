# 3.2 다중상수 SFE 정식화와 글로벌 검증

## 1. 동기: 단일상수의 반증과 설계 원칙

- 반증 포인트(요약):
  - 초기우주 $G$ 변화: 상수형 $G_{\text{eff}}/G = 1-\epsilon = 0.63$는 BBN/CMB의 $|\Delta G/G|\lesssim0.1$ 제약과 충돌.
  - SNIa 절대등급(Chandrasekhar 질량): $M_{\text{Ch}}\propto G^{-3/2}m_p^{-2}$이 $\sim5$배 변동 → 관측과 충돌.
  - 가변 상수: $\Delta\alpha/\alpha \sim \mathcal{O}(\epsilon)$ 가정은 원자시계·쿼이저 상한($\lesssim 10^{-8}$)과 충돌.

- 설계 원칙(무튜닝):
  - 미시(관성·운동학)는 $\epsilon_{\text{mass}}$로, 거시(중력)는 $\epsilon_{\text{grav}}$의 시간의존 함수 $\mu(a)$로 분리.
  - 게이지 결합은 상한만 허용($g_C\approx 0$)해 $\Delta\alpha/\alpha$ 제약을 만족.
  - 각 섹터는 “한 점 고정 → 전 구간 교차검증” 원칙으로 파라미터를 정함(피팅 금지, 사전 고정).

## 2. 수학적 정식화: 섹터 분리

### 2.1 질량/관성 섹터(미시)

$$\epsilon_{\text{mass}} \equiv g_B\,\frac{\mu}{\sqrt{\lambda}}, \qquad m_{\text{eff}} = m_0\,(1-\epsilon_{\text{mass}}).$$

무튜닝 고정: $\epsilon_{\text{mass}} \equiv 2\Omega_\Lambda - 1$.

파생 배율(모든 실험에서 추가 자유도 없이 동일):

$$\boxed{\;\frac{\tau_D^{\text{SFE}}}{\tau_D^{\text{STD}}}=\frac{1}{1-\epsilon_{\text{mass}}},\quad \frac{S_x^{\text{SFE}}}{S_x^{\text{STD}}}=\frac{1}{1-\epsilon_{\text{mass}}},\quad \frac{\tau_{\text{obs}}^{\text{SFE}}}{\tau_{\text{obs}}^{\text{STD}}}=\frac{1}{\sqrt{1-\epsilon_{\text{mass}}}}\;}$$

### 2.2 중력 섹터(거시)

$$\mu(a) \equiv \frac{G_{\text{eff}}(a)}{G_N} = 1 - \epsilon_{\text{grav}}\,S(a), \qquad S(\text{초기})\to 0,\; S(1)\to 1.$$

성장/렌징 방정식:

$$\ddot\delta + 2H\dot\delta - 4\pi G_N\,\mu(a)\,\rho_m\,\delta = 0, \qquad \Sigma(a) = \mu(a).$$

초기우주 정합: $a\ll1$에서 $S\to 0 \Rightarrow \mu\to 1$ (BBN/CMB 통과).

### 2.3 게이지 섹터(제약 상한)

게이지 운동항의 변형 $\mathcal{L}\supset + g_C\,\Phi\,\tfrac14 F_{\mu\nu}F^{\mu\nu}$은 $\alpha_{\text{eff}} = \alpha_0/(1-g_C\Phi)$를 유도. 원자시계·쿼이저·Oklo 제약을 보수적으로 만족시키기 위해 본 정식화에서는 $g_C\approx 0$ 상한만 명시한다.

## 3. 파라미터 고정 규칙(무튜닝)

- $\epsilon_{\text{mass}}$: $\Omega_\Lambda$로 1회 고정.
- $\epsilon_{\text{grav}}$: $S8$ 혹은 $f\sigma_8(z_0)$ 한 점으로 고정 → 이후 $f\sigma_8(z)$·$S8_{\rm lens}$ 전 구간 교차검증.
- $g_C$: 상한만 선언($\Delta\alpha/\alpha$ 제약 통과 범위 내).

## 4. 연역적 귀결과 반증 프로토콜

- 미시 3종(결맞음, LIGO 열잡음, 뮤온 수명)은 $\epsilon_{\text{mass}}$의 세 배율을 만족해야 함.
- 우주론(성장/렌징)은 $\mu(a)$로 적분한 $\{\sigma_8(0),\;S8,\;S8_{\rm lens},\;f\sigma_8(z)\}$를 $\le 1\sigma$ 정합으로 재현해야 함.
- 초기우주(BBN/CMB): $\mu(a\ll1)\to 1$를 수치적으로 확인.
- 가변 상수: $g_C\approx 0$ 가정 하에 $\Delta\alpha/\alpha$ 상한을 통과.

## 5. 간단 파이썬 검증 셀(무튜닝 배율 확인)

```python
Omega_L = 0.685
epsilon_mass = 2*Omega_L - 1
print({
  'epsilon_mass': epsilon_mass,
  'decoherence_scale': 1/(1-epsilon_mass),
  'LIGO_scale': 1/(1-epsilon_mass),
  'muon_tau_scale': 1/(1-epsilon_mass)**0.5
})
# 기대: {epsilon_mass: 0.37, decoherence/LIGO: ~1.587, muon_tau: ~1.259}
```

## 6. 비유와 직관(학부생용)

- 집 전체의 “메인 디머”가 관성(방 안 미끄러움)을, 계절 스케줄러가 중력(밖 풍경) 변화를 전담한다고 생각하자. 미시 변화(방 안)는 $\epsilon_{\text{mass}}$ 하나로, 거시 변화(밖)는 시간표 $\mu(a)$로 관리한다. 라디오 볼륨(게이지)은 건드리지 않는다($g_C\approx 0$).

## 7. 관측·실험 시그니처(튜닝 불요)

- 결맞음·LIGO·뮤온: 동일 배율 법칙(1.59, 1.59, 1.26) 재현.
- 성장·렌징: $\epsilon_{\text{grav}}$ 1점 고정 → 전 구간 $\le 1\sigma$ 일치.
- 초기우주: $\mu\to1$ 확인.
- $\alpha$ 불변: $g_C$ 상한 유지.

## 8. 순환 논리 차단

각 섹터의 고정은 서로 독립된 관측으로 1회만 수행하고, 나머지 지표는 연역적으로 예측한다. 동일 데이터로 고정과 검증을 동시에 하지 않는다.

## 9. 한계와 향후 과제

- $S(a)$의 구체적 정규화(예: $\int d\ln a\,\Omega_\Lambda(a)$ 누적)에 따른 민감도 분석.
- 뮤온 $g-2$의 정량치는 $\{m_\phi,\lambda\}$ 외부 제약 입력 필요.
- 은하 스케일(암흑물질) 대체는 본 최소 정식화의 범위를 넘어서며 별도 확장(φ-DM)에서 다룸.
